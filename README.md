# Python-Cybersecurity-Lab
Ce projet a été réalisé dans le cadre d'un projet académique en cybersécurité.

L'objectif est d'étudier et d'expérimenter différentes techniques liées à :

- La programmation Python
- L'administration système
- La surveillance d'activités sur un poste de travail
- La collecte et l'analyse d'informations système
- L'automatisation de tâches de sécurité

## Technologies utilisées

- Python 3
- FTP
- Cryptographie
- Gestion de fichiers
- Réseau
